# To-Do App

A simple and elegant To-Do List app built with SwiftUI and Core Data.

## Features

- Add/edit/delete tasks
- Mark tasks as complete
- Local data storage using Core Data
- Clean and responsive SwiftUI design

## Technologies

- Swift
- SwiftUI
- Core Data
- Combine
- MVVM architecture
